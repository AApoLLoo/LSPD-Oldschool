# Communication avec le DOJ

Ce document répertorie les protocoles de communication entre le LSPD et le Département de la Justice.

## Canaux de liaison

| Liaison | Description |
| :--- | :--- |
| **Liaison DOJ-LSPD** | Utilisé pour envoyer les éléments de preuve qui seront présentés lors du procès. |
| **Liaison PROC-LSPD** | Utilisé pour communiquer avec le procureur lors de la création de dossiers pénaux complexes. |

> **IMPORTANT :** Ne communiquer que les affaires en attente en respectant le format strict indiqué dans les modèles ci‑dessous.

---

## Modèles de communication (templates)

### 1. Signalement d'une affaire en attente

Ce format est à utiliser pour transmettre un dossier lorsqu'un suspect est en attente de procès ou d'avocat.

```text
NOM Prenom : {Nom du Suspect}
Telephone : {Numéro} :: Email : {Email} (du suspect)
Avocat : {Oui / Non} (si demandé spécifiquement, indiquer le nom de l'avocat)

Pour les crimes et délits suivants :
- {Crime 1}
- {Crime 2}
- {Délit 1}

Cordialement,
{Matricule} {Nom Prenom de l'Officier}
```
### 2. Mail Officiel au DOJ

Ce format est utilisé pour avertir le DOJ d'une nouvelle affaire sur le téléphone.
```text
Adresse destinataire : doj@oldschool.com

Objet : [Nom du plus gros chef d'inculpation]

Le suspect {Nom Prenom} accusé de :
- {liste des crimes / délits}

A demandé un avocat : {OUI / NON}
(Indiquer s'il demande un avocat spécifique ou un commis d'office)
```
